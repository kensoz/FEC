## スタック

- ⚡️ Next.js
- ⚙️ TypeScript
- 🎨 TailwindCSS
- 📑 ESLint + Prettier
- 🔌 Vitest + React Testing Library
- 🔩 GitHub Actions
- 💽 Firebase Cloud Firestore
- 🚀 Vercel
- 📦 Yarn



## 使用

##### インストール

```shell
yarn install
```

##### 開発

[http://localhost:8011](http://localhost:8011)で起動

```shell
yarn dev
```

##### ビルド

```shell
yarn build
```
